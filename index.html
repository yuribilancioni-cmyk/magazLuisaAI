📖 Funzionalità presenti
🏠 Corsie
• Categorie → linee → sublinee → prodotti
• Foto linea, badge prodotti mancanti
• + per segnare mancanti, × per nascondere
• Ordina scaffale con drag&drop
📦 Rifornimento
• Lista mancanti raggruppati per pallet
• Foto miniatura linea, scadenza colorata (verde/arancio/rosso)
• Selezione lotto specifico se più lotti disponibili
• 3 stati: Da prendere → 📦 Pronto → ✓ Caricato
• Conferma carico con quantità modificabile (max = lotto selezionato)
• Aggiornamento magazzino dal lotto selezionato
• FIFO automatico (scadenza più vicina prima)
🚚 Arrivo Merce
• 3 modalità: Normale / Pallet intero / Assortito
• AI: foto → riconosce prodotto + variante + EAN + scadenza + qtà
• Lista rilevati con miniatura, breadcrumb correggibile
• + Aggiungi clona ultimo prodotto (cambi solo variante)
• ✋ Inserimento manuale sempre disponibile
• Pallet selezionabile solo tra quelli esistenti
• Salva direttamente in Magazzino
🏭 Magazzino
• Pallet numerati, aggiungibili e rimovibili (solo se vuoti)
• Vista per Pallet → Fila → Prodotto
• Filtro per pallet con chip selezionabile
• Sposta lotto con rilevamento conflitti
📅 Scadenze
• Tutti i lotti ordinati per scadenza più vicina
• Rosso sotto 2 mesi, arancio sotto 4 mesi
✅ Controllo Mensile
• Lista linee da verificare a scaffale ogni mese
• Barra progresso, reset automatico ogni mese
🔍 Cerca
• Ricerca per linea, sublinea, variante
• Link diretto a Scaffale e Magazzino
📋 Task Extra
